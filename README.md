# Minecraft Server Build

## Версия: 1.0.0

### Как использовать:
1. Установите Git
2. Клонируйте репозиторий
3. Запустите скрипт обновления

### Структура:
- /server - серверная часть
- /client - клиентская часть
- /scripts - вспомогательные скрипты
- /backups - бэкапы мира
